README FIRST

Use website-brief.txt first.

The folder structure is intentional.

Project order is controlled by the number at the start of each folder name.

Do not display the leading numbers on the website.

The project name to display is generally the name written in brackets in the folder name.

Do not rename or move project folders.

Do not mix design iteration images into the normal gallery. If design iteration folders exist, show them in a separate Design Process section.

Use Helvetica normal and bold only throughout the website.

Keep the design minimal, clean, image-led and professional.
